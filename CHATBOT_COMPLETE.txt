    - MarianMT transformer models

    - 5-10ms processing

🚀 QUICK START (5 MINUTES)
================================================================================

STEP 1: INSTALL
  $ cd chatbot-engine
  $ python -m venv venv
  $ source venv/bin/activate
  $ pip install -r requirements.txt

STEP 2: RUN
  $ cd app
  $ uvicorn routes:app --reload --port 8000

STEP 3: TEST
  Open: http://localhost:8000/docs
  Or:   curl -X POST http://localhost:8000/chat \
        -H "Content-Type: application/json" \
        -d '{"message": "2 kg rice chahiye"}'

EXPECTED RESPONSE:
  {
    "user_message": "2 kg rice chahiye",
    "language": "Hindi",
    "translated_text": "I want 2 kg rice",
    "intent": "ORDER",
    "confidence": 0.93,
    "entities": {"product": "rice", "quantity": 2.0},
    "reply": "Great! I can process your order for 2kg rice.",
    "quick_replies": [...]
  }
