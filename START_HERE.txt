================================================================================
                         TROUBLESHOOTING
================================================================================

Frontend not loading?
  $ curl http://localhost:3001
  If no response, restart: npm run dev

Chatbot not responding?
  $ curl http://localhost:8000/health
  If no response, restart chatbot engine

Port already in use?
  $ lsof -i :3001    (for frontend)
  $ lsof -i :8000    (for chatbot)
  Then: kill -9 <PID>

Slow response?
  Normal! First load of chatbot downloads AI models (~500-1000ms)
  Subsequent requests are much faster (100-300ms)
